# LoveLorita

Acesta este un proiect de tip consolă scris în **C++**, reprezentând o aplicație de tip dating, ce folosește concepte de Programare Orientată pe Obiecte. 

# Funcționalități
  - Sistem de Utilizatori: Creare profil, afișare de utilizatori. Suportă diferite ierarhii de membri (`Gold`, `Premium`, `PremiumGold`)
  - Sistem de Match: Calcularea unui scor de compatibilitate între utilizatori pe baza vârstei, zodiei, preferințelor etc
  - Sistem de Mesaje: Utilizatorii cu statut avansat pot trimite mesaje altor utilizatori
  - Salvare/Încărcare în/din fișiere
  - Meniu Interactiv la Consolă

# Concepte Tehnice Folosite
- Clase, Obiecte, Constructori/Destructori.
- Moștenire (Virtuală și Multiplă).
- Supraîncărcarea operatorilor (`<<`, `>>`, `<`, `==`, `+`, etc.).
- Funcții și clase virtuale (Polimorfism).
- Șabloane (Templates) - de ex. `Afisator<T>`.
- STL (vector, set, list, map)
- Design patterns (Meniu Singleton).
